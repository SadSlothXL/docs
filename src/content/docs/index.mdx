Welcome to the IW4x documentation!

IW4x is a client for Call of Duty: Modern Warfare 2 (2009) that allows you to relive MW2 in a safe environment with support for dedicated servers and modding.

If you run into issues with IW4x or while following a Guide we are happy to help you out on our [Discord server](https://discord.com/invite/pV2qJscTXf)!
